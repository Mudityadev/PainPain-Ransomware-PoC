PainPain Ransomware PoC

A modern educational ransomware simulation with a realistic GUI, encryption engine, and safe C2 server demo — for ethical research only.

🚀 Get Started

🛠 Features

🔍 File Discovery

Scans directories recursively for target files.

🔐 AES Encryption

Encrypts files securely using a hardcoded key.

🧠 C2 Server

Simulates exfiltration of file and system data.

🖥 Modern GUI

Features a realistic ransomware interface.

📦 Usage

  1. pip install -r requirements.txt
  2. cp .env.example .env and configure secrets
  3. python c2_server_alt.py
  4. python main_v2.py -p "D:\\testDir" -e to encrypt
  5. python main_v2.py -p "D:\\testDir" -d to decrypt
  6. Use GUI and type bitcoin to trigger decryption

📁 Project Structure

  • main_v2.py — Core ransomware logic
  • wannacry_gui.py — GUI simulation
  • c2_server_alt.py — Simulated C2 server
  • discover.py & modify.py — File operations
  • .env.example & requirements.txt
