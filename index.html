Quick Start

  1. Install requirements: pip install -r requirements.txt
  2. Copy .env.example to .env and fill in your config.
  3. Start the C2 server: python c2_server/c2_server_alt.py
  4. Run the agent: python main.py -e (select target folder when prompted)
See the README for Docker, advanced usage, and more.

Module Highlights

  • CLI/Agent: main.py ‚Äî Entry point for encryption/decryption, CLI options, config loading.
  • Core Logic: ransomware/core.py ‚Äî Orchestrates key management, file discovery, encryption/decryption, C2 exfiltration, GUI launch.
  • GUI: ransomware/gui/main.py, ransomware/gui_main.py ‚Äî Tkinter GUI, timer, ransom note, payment instructions.
  • Crypto: ransomware/crypto/ ‚Äî Fernet-based encryption/decryption, key utilities.
  • Discovery: ransomware/discovery/ ‚Äî Recursively finds files, extension filtering.
  • Network: ransomware/network/ ‚Äî Sends exfiltration data to C2 server via HTTP POST.
  • C2 Server: c2_server/c2_server_alt.py ‚Äî Flask app, receives exfiltration data, logs events.
  • Tests: tests/ ‚Äî Unit and integration tests for all modules.
  • Sample Data: testDir/ ‚Äî Realistic test files for safe demonstration.
